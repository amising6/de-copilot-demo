# DE Copilot Demo

Metadata-driven data engineering automation.

Upload an STTM and automatically generate:

- Snowflake DDL
- Snowflake SQL
- Data Dictionary
- Technical Specifications
- Data Quality Rules

## Purpose

This prototype demonstrates how a single Source-to-Target Mapping (STTM) can become the source of truth for multiple engineering deliverables.

## Architecture

Frontend:
- Streamlit

Backend:
- Python
- Pandas

Outputs:
- Snowflake DDL
- Snowflake SQL
- Data Dictionary
- Technical Specifications
- DQ Rules

Deployment:
- Streamlit Community Cloud
